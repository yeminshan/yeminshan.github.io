1. 语言切换  
   插件下载 chinese language pack 安装重启即可切换成中文，
   ctl + shift + p （mac command + shift + p）输入 configure display language，进行切换。

2. 常用快捷键
   ctl + shift + p 命令面板
   ctl + p 快速打开 （文件）
   ctl + shift + n 新建窗口
   command + option + l 格式化

3. vue 文件使用大纲非常清晰

4. 集中管理 todo，使用 todo tree 插件

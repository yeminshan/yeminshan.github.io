## 文字
### 1. 段落文字除了最后一行左对齐， 其余两端对齐
```css
p{
    text-align:justify;
     text-justify:inter-ideograph;
}
```

### 2.文字超长省略
```css
.limit-text {
   max-width:1rem; /* 宽度必须 */
   overflow: hidden;
   white-space:nowrap;
   text-overflow:ellipsis;
}
```


